# T-120: 统一 iOS 初音色系修复

## 目标
将所有组件颜色统一为 iOS 标准色 + 初音主色

## 颜色规范（最终版）

| 用途 | 颜色 | Hex |
|:---|:---|:---|
| 主品牌/激活 | 初音 Teal | #39C5BB |
| 成功 | iOS Green | #34C759 |
| 警告 | iOS Orange | #FF9500 |
| 错误 | iOS Red | #FF3B30 |
| 信息 | iOS Blue | #007AFF |
| 文本主色 | iOS Label | #1C1C1E (亮) / #FFFFFF (暗) |
| 文本次要 | iOS Secondary | #8E8E93 |
| 背景 | iOS System | #F2F2F7 (亮) / #1C1C1E (暗) |
| 卡片背景 | iOS Secondary | #FFFFFF (亮) / #2C2C2E (暗) |
| 分隔线 | iOS Separator | #E5E5EA (亮) / #38383A (暗) |

## 需要修复的组件

### 1. CStatusDot
- online: `#52c41a` → `#34C759`
- warning: `#faad14` → `#FF9500`
- error: `#ff4d4f` → `#FF3B30`

### 2. CProgress
- success: `#52c41a` → `#34C759`
- exception: `#ff4d4f` → `#FF3B30`
- warning: `#faad14` → `#FF9500`

### 3. CSwitch
- inactive: `#bfbfbf` → `#E5E5EA` (亮) / `#3A3A3C` (暗)

### 4. CEmpty
- error 图标: `#ff4d4f` → `#FF3B30`

### 5. CAvatar（检查）
- 确保在线状态使用 `#34C759`

### 6. CTabs（检查）
- 激活状态使用 `#39C5BB`

### 7. CLoading（检查）
- 默认颜色使用 `#39C5BB`

### 8. BottomTab（检查）
- 激活状态使用 `#39C5BB`
- 未激活使用 `#8E8E93`
