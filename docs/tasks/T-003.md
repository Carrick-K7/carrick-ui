# T-095: carrick-ui 新增 CLogo 组件

## 目标
创建标准化的 Logo 组件，统一所有项目的 "Carrick + ProjectName" 品牌标识。

## 设计规范
```
[Carrick] [ProjectName]
  ↑39C5BB   ↑#666
  ↑Cascadia ↑Inter
```

| 属性 | 值 | 说明 |
|:---|:---|:---|
| Carrick 颜色 | `#39C5BB` | 品牌色，固定 |
| Carrick 字体 | `Cascadia Code` | monospace，固定 |
| Carrick 字重 | `700` (Bold) | 固定 |
| 项目名颜色 | `#666` | 固定 |
| 项目名字体 | `Inter` | sans-serif，固定 |
| 项目名字重 | `400` (Regular) | 固定 |
| 间距 | `10px` | `gap: 10px`，固定 |

## Props API
```ts
interface CLogoProps {
  project: string      // 项目名，如 "Dashboard", "Novel"
  size?: 'sm' | 'md' | 'lg'  // 默认 'md'
}
```

## 尺寸规范
| size | Carrick 字号 | Project 字号 | 用途 |
|:---|:---|:---|:---|
| sm | 20px | 14px | 移动端/紧凑布局 |
| md | 24px | 18px | 默认导航栏 |
| lg | 32px | 24px | 首页/展示页 |
