# T-002: 基于 Naive UI 重构组件库

## 目标
将 carrick-ui 从原生 Vue 组件重构为基于 Naive UI 的封装组件库。

## 变更内容
- [x] 安装 Naive UI 依赖
- [x] 创建 Carrick 主题配置（theme/carrickTheme.js）
- [x] 封装基础组件：
  - [x] CButton（基于 n-button）
  - [x] CCard（基于 n-card）
  - [x] CModal（基于 n-modal）
- [x] 保留自定义组件：
  - [x] TopNavbar（Naive 无现成导航）
  - [x] ImmersiveNav（需自定义）
- [x] 更新入口文件（index.js）
- [ ] 更新 Tech Spec
- [ ] 推送仓库

## 技术实现
- Naive UI 主题系统
- Cascadia Code 字体（微软开源）
- Carrick 品牌色 #39C5BB

## 组件清单

### 基于 Naive UI 封装
| 组件 | 基础 | 定制点 |
|:---|:---|:---|
| CButton | n-button | 默认使用 Carrick 品牌色 |
| CCard | n-card | 悬停动效、圆角 |
| CModal | n-modal | 圆角、阴影 |

### 完全自定义
| 组件 | 说明 |
|:---|:---|
| TopNavbar | 顶部导航栏（Naive 无此组件）|
| ImmersiveNav | 沉浸导航（游戏/创作类项目）|

## 使用方式

```javascript
import { createApp } from 'vue'
import { NConfigProvider } from 'naive-ui'
import { carrickThemeConfig, CButton, CCard } from '@carrick/ui-components'

const app = createApp(App)

// 使用 Naive UI 配置提供者
app.use(NConfigProvider, carrickThemeConfig)
```
