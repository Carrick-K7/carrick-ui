# T-112: 移动端导航组件 BottomTab（替代 T-111）

## 目标
创建移动端专用的底部标签栏组件，替代 Drawer 设计

## 设计理念（iOS Human Interface）
- **简洁**：只显示最重要的导航项
- **直觉**：底部操作，拇指友好
- **精致**：微动画、圆角、毛玻璃

## 设计规范

### 颜色（初音色系）
- 主色: #39C5BB（Carrick Teal）
- 激活: #39C5BB
- 未激活: #8E8E93（iOS灰色）
- 背景: rgba(255,255,255,0.9) + backdrop-blur
- 暗色背景: rgba(30,30,30,0.9)

### 布局
```
┌─────────────────────────────────┐
│      Content Area               │
│                                 │
├─────────────────────────────────┤
│ 🏠    📊    ⏰    👥    ☰  │
│总览   数据   定时   团队   更多│
└─────────────────────────────────┘
 ↑      底部固定，安全区适配
```

### 规格
- 高度: 64px（含安全区）
- 标签: 3-5 个（多了用"更多"收起）
- 图标: 24px
- 文字: 10px
- 圆角: 无（贴合底部）

### 动画
- 切换: 0.2s ease
- 激活图标: 轻微弹跳 scale(1.1) → scale(1.0)

## Props
```javascript
{
  navItems: Array<{name, path, icon, label}>,
  activeName: String,
  showMore: Boolean // 是否显示"更多"按钮
}
```

## Events
- @change: 切换标签时触发
