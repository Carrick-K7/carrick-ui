# T-100: 调整 Logo 和导航栏字体

## 目标
1. Carrick 字号加大（24px → 28px）
2. 项目名字体：Inter → **HarmonyOS Sans**
3. 项目名字号加大（18px → 20px）
4. 导航栏字体加大（14px → 15px）

## 字体引用
```html
<link href="https://fonts.googleapis.com/css2?family=HarmonyOS+Sans:wght@400;500;700&display=swap" rel="stylesheet">
```
或使用本地/CDN方式加载 HarmonyOS Sans

## 修改文件
- `carrick-ui/src/components/CLogo/CLogo.vue` - Logo 组件
- `carrick-ui/src/components/TopNavbar/TopNavbar.vue` - 导航栏组件  
- `carrick-ui/src/theme/carrickTheme.js` - 主题字体配置
